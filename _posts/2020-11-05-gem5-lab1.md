## Gem5: Creating a simple configuration script

因为Gem5的官方教程有时候有一些奇怪的错误，防止自己以后忘记了，在这里记录一下：

参考文档：http://www.gem5.org/documentation/learning_gem5/part1/simple_config/

教程中创建memory controller的代码如下：

```python
system.mem_ctrl = DDR3_1600_8x8()
# 指定了memory controller控制的memory地址的范围
system.mem_ctrl.range = system.mem_ranges[0]
system.mem_ctrl.port = system.membus.master
```

但是实际运行中，会报错：

```python

    % (self.__class__.__name__, attr))
AttributeError: Class DDR3_1600_8x8 has no parameter port
```

原因暂时查不明白。官方其实提供了可运行脚本simple.py在 `configs/learning_gem5/part1/simple.py`，它是这么写的：

```python
system.mem_ctrl = MemCtrl()
system.mem_ctrl.dram = DDR3_1600_8x8()
system.mem_ctrl.dram.range = system.mem_ranges[0]
system.mem_ctrl.port = system.membus.master
```

不太一样，Hmmm

但是改了之后代码运行就正常了
